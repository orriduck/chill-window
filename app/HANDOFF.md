# 窗景 · 专注列车（交接文档）

一个以"火车侧窗风景"为核心体验的专注 App。窗外是程序化生成的流动风景，完整模拟"检票上车 → 始发站停靠 → 发车缓加速 → 沿途巡航 → 进站停靠（休息）→ 再次发车 → 抵达终点"的旅程节奏。

## 技术栈

- React 18 + TypeScript + Vite + Tailwind CSS 3.4 + shadcn/ui
- 渲染：Canvas 2D（场景）+ WebGL（铅笔素描后处理）
- 音效：Web Audio API 程序生成（无音频资源文件）
- 纯前端，无后端；当前无数据持久化（刷新即丢失）

## 构建运行

```bash
npm install
npm run dev    # 开发
npm run build  # 构建到 dist/
```

## 风景引擎架构（scenery.ts）

**分层视差**（速度比例）：天空/云 → 远山(0.045) → 中景(0.17) → 近景地面(0.5) → 前景(1.0)

- **远山/中景**：离线渲染成 2048px 可无缝循环条带（整数倍谐波叠加保证循环），双场景滚动切换
- **场景过渡**：新场景条带从右缘驶入覆盖旧场景，远山/中景错峰进入；装饰物按新场景类型提前生成
- **5 类场景**：田野 / 森林 / 山地 / 河畔 / 小镇，各有独立的远山+中景条带生成器
- **装饰物系统**：世界坐标锚定（`Decor.x`），随世界滚动，27 种素材按场景权重随机刷出；每种都有昼夜自适应配色
- **树 4 形态**：阔叶（多叶团分层）/ 针叶（塔形裙边）/ 垂枝柳 / 枯枝 + 树丛（近树遮挡远树）
- **站台**：世界坐标锚定约 450m 长度，含雨棚/站房/出站口/站名牌/长椅/垃圾桶/候车人（带行李箱）
- **其他**：隧道（随机穿越+灯带）、接触网电线杆(780px间距+绝缘子)、邻线铁轨(钢轨/枕木/道砟)、近景灌木篱动态层、飞鸟、云

**关键参数**：`MAXX=430`(巡航px/s) `P_FAR/P_MID/P_NEAR`(视差比) `accel`(加减速率)

## 铅笔素描渲染（pencil.ts）

参考 chartogne-taillet.com 的 shader 思路移植为 2D 后处理：
1. Sobel 边缘检测 → 轮廓线（阈值 0.16-0.42，只勾明暗交界）
2. 双层交叉排线（0.55/-0.95 弧度两角度），按暗度分级 smoothstep(0.52,0.88) / (0.72,0.97)
3. Perlin 噪声块状打断笔触 → 手绘感
4. 纸色/铅笔色双色调 + 纸张纹理 + 噪点 + 暗角；夜晚自动反相（深纸浅笔）

注意：天空调色板已提亮（`skyTop: #7ec3ee`），保证转素描后天空留白。

## 旅程状态机（Home.tsx）

`setup → ride ⇄ dwell → done/abort`；经停站=休息（dwell，不计入专注时长）；"中途下车"=abort 结算。进站前 16s 开始减速停靠。

## 设计约束（别踩坑）

- 素描模式依赖"亮部留白"：新增景物时亮部要够亮，否则会被排线铺满
- 所有新素材必须同时提供昼夜两套配色（通过 `pal.night` 分支）
- 装饰物间距通过世界坐标控制，不要用屏幕坐标（否则加减速时会露馅）
- 站台相关绘制必须世界锚定（`platformWx - worldX`），不能跟随屏幕
